# ChineseSubFinder Panel

管理面板

## 安装

```bash
npm install
```

### 运行

```bash
npm run dev
```

### 构建

```bash
npm run build
```

## 配置文件说明

| 文件名           | 环境     |
| ---------------- | -------- |
| .env.development | 开发环境 |
| .env.production  | 正式环境 |

配置项说明：

| key         | value             |
| ----------- | ----------------- |
| BACKEND_URL | 后台 API 服务地址 |
